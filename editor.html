Let's see what we can do

This editor has some extra commands. They have to be written inside the opening < ! ! ! // without the spaces and closing ! ! ! > // without the spaces tag. Right before the element you want to manipulate. Commands that are currently supported:

  • float-left // floats image left and text will flow around the image
  • float-right // floats image right and text will flow around the image
  • width-[number] // sets the width of an image relative to the column width. accepts numbers from 10 - 140 eg. width-40
  • width-max // sets image to max width of extended column
  • float-aside // floats the content aside the main column
